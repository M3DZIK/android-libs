# Add to your project

Add the library as a dependency to your android project.

```kotlin
dependencies {
    // Compose
    implementation("dev.medzik.android:compose:1.6.1")
    // Crypto
    implementation("dev.medzik.android:crypto:1.6.1")
    // Utils
    implementation("dev.medzik.android:utils:1.6.1")
}
```
